## Training Plots (`training/{model_name}/`)

One subfolder per trained YOLO26 model. Each contains 7 standard Ultralytics plots.

### Models

| Subfolder | Architecture | mAP50 | Notes |
|-----------|-------------|:-----:|-------|
| `y26n_vanilla_local/` | YOLO26n | **0.521** | Best overall + fastest |
| `y26s_vanilla_local/` | YOLO26s | 0.506 | Standard pretrained |
| `y26m_vanilla_local/` | YOLO26m | 0.509 | Largest; best E2E |
| `y26s_nopretrained/` | YOLO26s | 0.511 | Ablation: scratch training |
| `y26s_noaug/` | YOLO26s | 0.465 | Ablation: no augmentation |

### Plot Types (same in each subfolder)

| File | Description |
|------|-------------|
| `confusion_matrix.png` | Raw confusion matrix (4 classes + background) |
| `confusion_matrix_normalized.png` | Row-normalized confusion matrix (recall per class) |
| `results.png` | Training curves: box loss, cls loss, dfl loss, mAP50, mAP50-95 |
| `BoxPR_curve.png` | Precision-Recall curve per class |
| `BoxP_curve.png` | Precision vs confidence threshold |
| `BoxR_curve.png` | Recall vs confidence threshold |
| `BoxF1_curve.png` | F1 vs confidence threshold |

### Key observations from training plots

- **`y26s_noaug/results.png`** — Training curves show clear overfitting at epoch 6;
  validation mAP50 degrades immediately after. Augmentation is essential.

- **`y26s_nopretrained/results.png`** — Training from scratch requires more epochs (57
  vs 21–38 for pretrained) but converges to a similar final mAP50.

- **`confusion_matrix_normalized.png` (all models)** — B2↔B3 off-diagonal is the
  dominant confusion in all models, confirming the irreducible visual ambiguity
  between these two maturity classes.
